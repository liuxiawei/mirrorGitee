这是一个自动将 Gitee 仓库镜像到 Gitea 的 Shell 脚本。支持批量同步所有仓库，可以设置跳过特定仓库，并具有良好的错误处理机制。

## 功能特性

- 自动同步 Gitee 所有仓库到 Gitea
